Data Asset class¶

class starfish.asset.DataAsset(metadata, did=None, data=None, metadata_text=None)¶

Bases: starfish.asset.asset_base.AssetBase

File asset can be used manage a data asset on the ocean network

Parameters

  • metadata (None or dict) – Dictionary metadata to provide for the asset.

  • did (None or str) – Optional did of the asset if it’s registered

  • data (str) – Optional data of the asset, this can be str or bytes
